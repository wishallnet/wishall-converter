Kruti Dev Converter Guide

1. Why is Kruti Dev Important?

Kruti Dev is the standard font for Government Typing Exams (SSC, State Exams, MP Patwari, UP Police) in India. It is a legacy font that requires specific Remington keyboard mapping. While Unicode (Mangal) is the web standard, many legal and government offices still operate on Kruti Dev 010.

2. The 'Smart Quotes' Problem

A major issue users face is when typing in MS Word. Word automatically changes straight quotes (') to curly "Smart Quotes" (‘ ’). In Kruti Dev, the quote key is used for the character 'Sh' (श) and 'Shha' (ष).

When these quotes curl, the underlying code changes, and ordinary converters translate them incorrectly. For example, 'Shaam' (शाम) might turn into 'Shhaam' (षाम) or garbage text.

3. Wishall's Smart Solution

Wishall Smart Hindi Editor solves this instantly. Our algorithm detects these curled quotes and reverts them to their correct legacy code before conversion.

कृति देव कनवर्टर गाइड

1. कृति देव क्यों महत्वपूर्ण है?

कृति देव भारत में सरकारी नौकरी की टाइपिंग परीक्षाओं (SSC, राज्य परीक्षा, पटवारी) का मानक फोंट है। यह रेमिंगटन कीबोर्ड लेआउट का उपयोग करता है। हालांकि इंटरनेट पर यूनिकोड (मंगल) चलता है, लेकिन अदालतों और सरकारी दफ्तरों में आज भी कृति देव 010 का बोलबाला है।

2. 'स्मार्ट कोट्स' की समस्या

एमएस वर्ड (MS Word) में टाइप करते समय सबसे बड़ी समस्या 'कोट्स' की आती है। वर्ड अपने आप सीधे कोट्स (') को घुमावदार कोट्स (‘ ’) में बदल देता है।

चूंकि कृति देव में 'श' और 'ष' के लिए कोट्स वाली की (Key) का उपयोग होता है, इसलिए यह ऑटो-फॉर्मेटिंग आपके शब्दों को बदल देती है। साधारण कनवर्टर इसे समझ नहीं पाते और 'शाम' को 'षाम' बना देते हैं।
